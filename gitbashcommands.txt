ls - show all files in structure
cd [folder] - go into file
git clone [url] - clone reposetory to location
git status - show status of files in folder
git add . - adds files to list to do something
git commit -m ["commit message"] - marks files to commit
git push - pushes it though and commits it
git checkout - b ["name"] - create a branch
git push - set-upstream origin Jacobs_branch - push the current branch and set the remote as upstream
git branch - gives a list of branches
git checkout [branch] - move to branch or main
